### Task 1: Fix x-default consistency

**Problema:** `x-default` punta a URL senza locale (es. `/chi-siamo/`) che triggera 308 redirect → /it/ — link equity sprecata. Due implementazioni su tre sono sbagliate.

**Files:**
- Modify: `src/lib/i18n/locale-metadata.ts:53-56`
- Modify: `src/app/[locale]/page.tsx:146-153`

**Step 1: Fix locale-metadata.ts**

Apri `src/lib/i18n/locale-metadata.ts`. Sostituisci le righe 53-56:

```ts
// PRIMA (sbagliato — punta a /chi-siamo/ che fa 308 → /it/chi-siamo/):
  // x-default: root Italian path (no locale prefix), with translated canonical slug
  const itSlug = translateSlug(canonicalSlug, 'it');
  const itPath = restPath ? `/${itSlug}/${restPath}/` : `/${itSlug}/`;
  languages['x-default'] = `${BASE}${itPath}`;
```

Con:

```ts
// DOPO (corretto — /en/{slug} è una pagina reale, nessun redirect):
  // x-default: English path — /en/ is a real page with no redirect.
  // The bare canonical path (e.g. /chi-siamo/) triggers a 308 → /it/ leaking link equity.
  const enSlug = translateSlug(canonicalSlug, 'en');
  const enPath = restPath ? `/${enSlug}/${restPath}/` : `/${enSlug}/`;
  languages['x-default'] = `${BASE}/en${enPath}`;
```

**Step 2: Fix homepage x-default in [locale]/page.tsx**

Apri `src/app/[locale]/page.tsx`. Trova la riga (circa 153):

```ts
  languages['x-default'] = `${BASE_URL}/`;
```

Sostituisci con:

```ts
  languages['x-default'] = `${BASE_URL}/en/`;
```

Nota: `/en/` è servita da `app/[locale]/page.tsx` con locale="en" — pagina reale, nessun redirect.

**Step 3: Verifica build TypeScript**

```bash
cd /mnt/disco_secondario/GeoTapp-EcoSystem/geotapp-site
npx tsc --noEmit 2>&1 | tail -10
```

Expected: no errors.

**Step 4: Commit**

```bash
git add src/lib/i18n/locale-metadata.ts src/app/\[locale\]/page.tsx
git commit -m "fix(seo): x-default → /en/ on all pages, removing 308-redirect targets"
```
